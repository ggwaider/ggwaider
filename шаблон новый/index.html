После чего мною было принято решение загрузить свои мозги по полной, для этого я записался на курсы в по HTML CSS и JS в школу Stormnet. Занятия в которой стартовали с 11.03.2021г. Дабы на курсах не быть полным нулем я предпринял всяческие меры по изучения всего вышеуказанного сам. И вот данный сайт тому пример. Впринципе ничего трудного.
